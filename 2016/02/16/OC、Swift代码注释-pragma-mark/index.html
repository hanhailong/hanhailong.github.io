OC、Swift代码注释#pragma mark

2016-02-16
IOS, OC, Swift

以前在我们写OC代码的时候，经常通过下面语法来为代码添加注释：

#pragma mark – Initialization 
code here...
#pragma mark – Table Managementmore 
code here...

但是，如果是Swift代码，这种写法就不好使了，下面给出Swift代码注释，Swift代码注释有三种:

  1. MARK

    // MARK: - Initialization
code here...
// MARK: - View Managementmore 
code here...

  2. TODO

    // TODO: - XXXXXXXX
code here...

  3. FIXME

    // FIXME: - XXXXXXXX
code here...
